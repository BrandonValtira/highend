# High-End Web Techniques

A project exploring modern, high-end web development techniques including parallax scrolling and other advanced visual effects.

## Features

- **Parallax Scrolling**: Multiple layers with different scroll speeds for depth effect
- **Performance Optimized**: Uses `requestAnimationFrame` and throttling for smooth 60fps
- **Responsive Design**: Works across all device sizes
- **Modern CSS**: Gradient backgrounds, smooth animations, and transitions

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Scroll to see the parallax effects in action

## Techniques Demonstrated

- Parallax scrolling with multiple layers
- CSS animations and transitions
- Performance optimization with requestAnimationFrame
- Responsive design patterns
- Modern gradient backgrounds

## Browser Support

Works in all modern browsers that support:
- CSS transforms
- requestAnimationFrame
- ES6 JavaScript
